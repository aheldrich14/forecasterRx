About forecasterRx
=================
Predictive analytics platform for retail pharmacy operations.

Main Features
=================

* Demand Forecasting

    - Hierarchical forecasting for medication inventory.
![img](demand_forecasting/data/sample.png)


Getting Started
=================

1. Clone repository.
1. `pip install -r requirements.txt`
1. `cd forecasterRx`
1. `python app.py`
